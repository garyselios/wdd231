Website Site Plan

CinemaFinder - Movie & TV Series Search Engine

1. Site Name

CinemaFinder

This name was chosen because it clearly describes the site's purpose: helping users find and discover movies and TV series. "Cinema" represents the film industry, and "Finder" indicates the search and discovery functionality.

2. Site Purpose

The CinemaFinder website provides an interactive platform for users to:

  • Search and discover movies and TV series using real-time data from TMDB API
  • Browse new releases, popular content, and personalized recommendations
  • Filter content by genre, release year, rating, and other criteria
  • Save favorite movies to a personal watchlist using Local Storage
  • Get detailed information about films

3. Scenarios

Target audience questions that will drive website content:

  • "What are the best action movies released in the last year?"
  • "How can I find movies similar to my favorites?"
  • "Where can I save movies to watch later?"
  • "What new TV series are coming out this month?"

4. Color Scheme

The color scheme is inspired by cinema and dark mode viewing experiences:

Primary Color

#0d253f (Dark Blue)

Used for: Headers, navigation bar, footer

Secondary Color

#01b4e4 (Light Blue)

Used for: Buttons, links, highlights, accents

Accent Color

#90cea1 (Soft Green)

Used for: Special elements, success messages, ratings

Background

#121212 (Dark Gray)

Used for: Main background, card backgrounds

Text Color

#ffffff (White)

Used for: Primary text on dark backgrounds
