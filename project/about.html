About CinemaFinder

Our Story

CinemaFinder was created as a final project for WDD231 - Web Frontend Development. The goal was to build a dynamic, responsive web application that integrates real-time data from an external API while demonstrating proficiency in HTML, CSS, and JavaScript.

The website uses The Movie Database (TMDB) API to fetch up-to-date information about movies, including popular titles, trending films, and detailed metadata. Users can search for movies, filter results by genre, year, and rating, and save their favorites using browser localStorage.
